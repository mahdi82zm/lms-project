# LMS Project Requirements

## Project Title
Simple Learning Management System (LMS)

## Description
This project is a web-based learning management system where instructors can create courses and students can enroll in them, access course content, and take quizzes.

## Main Roles
- Admin
- Instructor
- Student

## Functional Requirements
1. Users can register and log in.
2. The system supports different roles (admin, instructor, student).
3. Instructors can create, edit, and delete courses.
4. Students can view available courses.
5. Students can enroll in courses.
6. Instructors can upload course content.
7. Students can access course materials.
8. Instructors can create quizzes.
9. Students can participate in quizzes.
10. The system shows quiz results.

## Non-Functional Requirements
- Simple and user-friendly UI
- Secure authentication
- Responsive design for desktop and mobile
- Clean API communication between frontend and backend
